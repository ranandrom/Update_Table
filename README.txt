使用该程序前请仔细阅读：

1.该程序为V1.6.0版本，主要实现生信更新表的数据处理功能，最后会在指定目录下生成一个Excel表和一个tsv格式文档，其tsv格式文档内容与该Excel表内容一致。

2.该程序只支持Excel的xlsx格式版本。

3.该程序的运行环境是linux系统+java1.8版本以上。

4.该程序的使用方法为：java -jar Update_Table_2.jar xxxx(输入参数)
其中输入参数为：
(1).【-P/-p 指定的有效路径】：如果不输入时，默认为：/Src_Data1/analysis/Ironman/
(2).【-C/-c 0或1】：输入0或1，0代表覆盖之前的同名Excel表，1代表在原有的Excel表上进行追加，默认为追加模式。
(3).【-O/-o 指定的有效路径】：生成的文件存放路径，默认为：./Ironman。
(4).【-U/-u 0或1】：输入0或1，0代表全部上传，1代表只上传更新文件，默认为0，即全部上传。
