使用该程序前请仔细阅读：

1.该程序为V1.3.0版本，主要实现生信更新表的数据处理功能，最后会在当前目录下生成一个tsv格式文档，其内容与该Excel表内容一致。

2.该程序只支持Excel的xlsx格式版本。

3.该程序的运行环境是linux系统+java1.8版本以上。

4.该程序的使用方法为：java -jar Update_Table_2.jar xxxx(输入参数)
其中输入参数为：
(1).【-P/-p 指定的有效路径】：如果不输入时，默认为：/Src_Data1/analysis/Ironman/
(2).【-C/-c 带路径的文件名】：输入0或1，0代表覆盖之前的同名Excel表，1代表在原有的Excel表上进行追加，默认为追加模式。
(3).【-O/-o 带路径的文件名】：需要生成的tsv格式文件名，默认为：./Ironman。

5.该版本比上版本增加了以下几点：
（1）.增加了先自动复制数据汇总目录下的目前最新的Excel表，把其中文件命名的日期更新为当天日期，若没有该目录，则自动新建。
（2）.新增以下3列：：
1）：Sample QC：样本质检结果。取质检中最差的一项作为该样本的质检结果。
2）：Failed QC Detail：将Failed的项目列出来。
3）：Warning QC Detail：将Warning的项目列出来。
